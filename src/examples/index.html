AppServer Examples

These examples cover GH Documents that might have additional information, or aren't handled by the /view endpoint.

beam GH definition demonstrating beam analysis with adjustable length, position, and toggle parameters. Shows how to work with numeric inputs and download results.
bendy Example which uses the Kangaroo solver for interactive physics simulation.
convert File conversion tool that accepts STL, OBJ, and FBX files and converts them to Rhino 3DM format using Grasshopper.
docString Example which shows how to return a rhinoDoc from Grasshopper and parse it in the browser.
metaball table Example which uses interaction to move input points around for metaball generation.
panels This example shows how to use numeric output to add more information in the UI.
spikyThing GH definition that uses SubD components to create a spiky ball. Output is returned as a b64 Draco compressed mesh and is decompressed with rhino3dm. Includes timing extraction code.
test-working Example demonstrating value list input with a dropdown selector for different numeric values.
valueList Example which shows how to work with a value list as input.
topological-optimization Advanced topological optimization example using multiple parameters (thickness, min/max radius, square, strut size, segment, links, corners, smooth) with geometry input support.
