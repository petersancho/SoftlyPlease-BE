STEP 3: OPEN POWERSHELL AS ADMINISTRATOR
================================================================================

ON THE AZURE DESKTOP:
1. Look at BOTTOM LEFT corner
2. Click the WINDOWS FLAG logo
3. Type: powershell
4. Wait for "Windows PowerShell" to appear

WHEN YOU SEE THIS:
+-------------------+
| [BLUE BOX ICON]   |
| Windows PowerShell| <-- RIGHT-CLICK THIS
+-------------------+

WHEN YOU RIGHT-CLICK, SEE THIS MENU:
+-------------------+
| Open              |
| Run as            |
|  Administrator    | <-- CLICK THIS OPTION
+-------------------+

WHEN IT ASKS:
+-----------------------------+
| Do you want to allow this  |
| app to make changes to     |
| your device?               |
|                           |
| [Yes]                      | <-- CLICK YES
+-----------------------------+

SUCCESS: PowerShell window opens with this at the top:
+-----------------------------+
| Administrator: Windows     |
| PowerShell                 |
+-----------------------------+

STEP 4: NAVIGATE TO SCRIPTS FOLDER
================================================================================

POWERSHELL WINDOW LOOKS LIKE THIS:
+-----------------------------+
| PS C:\Users\Administrator> | <-- This is the prompt

cd C:\Users\Administrator\Desktop\SoftlyPlease\scripts\azure

| scripts\azure>                                    | <-- New prompt
